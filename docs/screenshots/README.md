## How to Capture Screenshots

### macOS
1. Open the application at http://localhost:5001
2. Use `Cmd + Shift + 4` to capture a selection
3. Save to this folder with the correct filename

### Windows
1. Open the application at http://localhost:5001
2. Use `Windows + Shift + S` to capture a selection
3. Save to this folder with the correct filename

### Using Browser DevTools
1. Open Chrome DevTools (F12)
2. Click the device toolbar icon
3. Set a consistent viewport (e.g., 1280x800)
4. Use the screenshot feature in DevTools

## Screenshot Guidelines

- Use a consistent viewport size (1280x800 recommended)
- Capture in dark mode (the default theme)
- Include realistic sample data
- Blur or remove any sensitive information
- Save as PNG format for best quality
